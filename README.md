# Vita Development Suite Ext

[Vita Development Suite Startup Guide](https://github.com/Vita-Development-Suite) with some of my personal extentions, mainly for userland applications.
Required for building complex system applications such as [NetStream](https://github.com/GrapheneCt/NetStream).

# Installation

Follow instructions in the [Vita Development Suite Startup Guide](https://github.com/Vita-Development-Suite/doc), but use components from the [Releases](https://github.com/GrapheneCt/Vita-Development-Suite-Ext/releases/) of this repository.
